# Ordem dos Seletores no CSS

- Nós aprendemos sobre o seletor de id e classe;
- E se a tag estiver com id e uma classe, o que acontece?
- Como nas maneiras de adicionar CSS, também tem uma ordem;
- Que é: id > class > seletor de tag;
- Então o id vai vencer todos os outros, utilize isso ao seu favor.
- Regras que não entram em conflito são aplicados normalmente.
