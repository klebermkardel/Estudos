# Algoritmos - Introdução

Algoritmos != Algarismos

"**Algoritmos** são conjuntos de passos **finitos** e **organizados** que, quando executados, resolvem um determinado **problema**."

*ALGORITMOS - Lógica para Desenvolvimento de Programação de Computadotes: José Augusto N. G. Manzano*

Exemplo de algoritmo (segue uma rotina padrão):

```
Algoritmo AtravessarRua
    Olhar para a direita
    Olhar para a esquerda
    Se estiver vindo carro
        Não Atravesse
    senão
        Atravesse
    Fim-Se
Fim-Algoritmo
```

Exemplo de não algoritmo (não segue uma rotina padrão):

```
Algoritmo AtravessarRua
    Atravesse
    Se estiver vindo carro
        Olhar para a direita
    senao
        Olhar para a esquerda
    Fim-Se
    Não Atravesse
Fim-Algoritmo
```

## Algoritmos Computacionais

São passos a serem seguidos por um **módulo processador** e seus respectivos **usuários** que, quando exececutados na ordem correta, conseguem **realizar** determinada **tarefa**.

- Módulo Processador: tudo aquilo que pode efetuar processamento
- Usuário: quem está utilizando o programa
- Realizar tarefa: resolver algum problema, sanar uma necessidade

Lógica de Programação -> Linguagem de Programação -> Sistema Completo
