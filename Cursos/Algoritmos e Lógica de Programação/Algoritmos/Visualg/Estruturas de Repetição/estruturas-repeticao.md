# Visualg - Estruturas de Repetição

## Estrutura Enquanto (while)

Sintaxe:

```
Enquanto condicao faca
    bloco
FimEnquanto
```

Exemplo genérico:

```
mao <- 0
Enquanto (mao <= 5) faca
    troca
    mao <- mao + 1
FimEnquanto
```

Exemplo vida real:

```
Enquanto não arrumar o quarto faca
    castigo
FimEnquanto

liberado do castigo
```
