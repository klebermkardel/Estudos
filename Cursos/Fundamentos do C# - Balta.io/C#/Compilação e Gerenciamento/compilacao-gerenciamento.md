# Fundamentos do C#

## Compilação e Gerenciamento

### Definições

- A Microsoft possui outras linguagens como VB.NET, F# e até Cobol.NET
- Todas elas precisam ser mantidas
    - Manutenção de código mesmo, correções de bugs e afins
- Quando o ecossistema de desenvolvimento Microsoft foi criado ele foi pensado desta forma
    - Suporte a várias linguagens
- Se cada linguagem tiver um gerenciador, isto seria muito complicado
- Compiladores ficam separados
    - Um para cada linguagem (ou mais de uma linguagem)
- A compilação inicial gera um código intermediário
 - Intermediate Language (IL)

 *** IL: Sigla para Intemediate Language ou linguagem intermediária ***
