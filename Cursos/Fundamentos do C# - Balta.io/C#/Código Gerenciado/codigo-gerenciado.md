# Fundamentos do C#

## Código Gerenciado

### Definições

- Significa que sua execução depende de um gerenciador
- Também conhecido como Runtime
    - No caso do C# este Runtime se chama CLR ou CLR Runtimr
- Gerencia memória, segurança, entre outros recursos básicos

*** CLR: é a sigla para Common Language Runtime ou gerenciador de linguaguem comum. Entenderemos este nome mais adiante. ***

- Em C/C++ por exemplo, fazemos alocação de memória manualmente
- Sabe a famosa tela azul do Windows?
- Permite interagirmos menos ou quase não interagir com recursos da máquina
