# Fundamentos do C#

## Lançamentos

### Definições

- .NET Framework
    - Não possuia uma data de lançamento programada
- .NET Core
 - Lançamentos (releases) semestrais
- .NET 5 e futuros
    - Lançamentos Anuais

## LTS

### Definições

- Sigla para Long Term Support
    - Suporte de longa data
- .NET Framework
    - Todos eram
- .NET Core
    - Versões com final 1
        1.1, 2.1, 3.1...
- .NET 5 e futuros
    - Versões maiores (Major Version)
- Opote sempre por LTS para projetos em produção
