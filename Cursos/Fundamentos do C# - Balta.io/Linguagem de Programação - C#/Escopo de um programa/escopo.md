# Fundamentos do C#

## Linguagem de Programação - C#

### Escopo de um programa

#### Definições

- Importações
- Namespace
- Classe
- Método principal

#### Exemplo

```
using System; // using => Importações

namespace MeuApp // namespace => Divisões Lóficas
{
    class Program // class => classes // Função principal do programa
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World");
        }
    }
}
```
