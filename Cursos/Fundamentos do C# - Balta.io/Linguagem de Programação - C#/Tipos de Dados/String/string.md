# Fundamentos do C#

## Linguagem de Programaçã - C#

### Tipos de Dados
 
#### String

##### Definições

- Armazena uma cadeira de caracteres
- Podemos pensar nele como uma lista de `char`
= Definido pela palavra reservada `string`
- A atribuição de um valor string é dada por aspas duplas
- string (?)
    - Uma cadeia de caracteres

##### Exemplo

```
string saudacao = "Olá, como vai?";
```
