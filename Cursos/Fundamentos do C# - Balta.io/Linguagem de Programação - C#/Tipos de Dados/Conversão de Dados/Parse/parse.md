# Fundamentos do C#

## Linguagem de Programaçã - C#

### Tipos de Dados
 
#### Conversão de Dados

##### - Parse

###### Definições

- Método presente em todo tipo primitvo
- Usado para converter um caractere ou string para um tipo qualquer
- Caso haja alguma incompatibilidade, gera um erro

###### Exemplo

```
int inteiro = int.Parse("100");
```
