#### Valores Padrões

##### Definições

- Todo tipo built-in, já possui um valor padrão
- Se nenhum valor for informado, seu valor padrão será utilizado
    - int => 0
    - float => 0
    - decimal => 0
    - bool => false
    - char => '\0'
    - string => ""
