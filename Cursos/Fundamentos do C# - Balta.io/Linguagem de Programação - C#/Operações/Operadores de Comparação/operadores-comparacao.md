# Fundamentos do C#

## Linguagem de Programaçã - C#

### Operações

#### Operadores de Comparação

##### Definições

- Podemos comparar qualquer tipo de dado
    - Números, strings, bytes
- A comparação SEMPRE retorna verdadeiro ou flaso
    - True ou false

- Igual: ==
- Diferente: !=
- Maior que: >
- Menor que: <
- Maior ou igual a: >=
- Menor ou igual a: <=

##### Exemplo

```
int x = 25;
x == 0; // False
x != 0; // True
x > 0; // True
x < 0; // False
x <= 0; // False
x >= 0; // True
```
