# Introdução aos Layouts no HTML 5

Neste resumo, vamos explorar os layouts no HTML5. Embora seja uma funcionalidade relativamente nova, o conceito e a teoria por trás dos layouts não são novos. Analisaremos duas páginas conhecidas no Brasil para entender como os layouts funcionam: o site da globo.com e o site da Caixa.

O site da globo.com utiliza recursos para organizar seu conteúdo, como um logotipo no canto superior esquerdo, um campo de pesquisa, temperatura e um menu com informações sobre notícias, esportes e entretenimento. As notícias são apresentadas em diferentes seções, com as mais importantes em destaque. O rodapé contém informações sobre direitos autorais e privacidade.

O site da Caixa também possui elementos semelhantes, como um logotipo no canto superior esquerdo, menus na parte superior com informações úteis e um campo de pesquisa. A Caixa tem um menu lateral para informações sobre loterias e programas sociais, além de informações adicionais no lado direito.

Na maioria dos sites, observamos uma organização semelhante, com o logotipo no início, menus na parte superior ou lateral e o conteúdo no meio. No HTML5, essa organização dos elementos é chamada de layout.

Ao pesquisar no Google sobre HTML5, encontramos vários layouts pré-definidos que podem ser utilizados. Podemos ter uma página com cabeçalho e rodapé, um menu de navegação chamado "nav" e informações no meio. Também podemos escolher organizar a página apenas com cabeçalho, menu lateral ou um menu adicional na lateral oposta, seguido do rodapé. A flexibilidade na escolha da organização é uma vantagem do HTML5.

O HTML5 introduz novas tags para determinar o significado de cada elemento na página. As principais tags para organizar o conteúdo são: cabeçalho (tag `<header>`), menu de navegação (tag `<nav>`), seção (tag `<section>`) e rodapé (tag `<footer>`). Podemos ter apenas um cabeçalho, um menu e um rodapé, mas várias seções, como entretenimento, notícias e esportes.

Vamos fazer um teste com a página "Oi mundo" que criamos anteriormente. Dentro da tag `<body>`, adicionaremos os elementos necessários: `<header>`, `<nav>`, `<section>` e `<footer>`. Podemos salvar o arquivo e visualizá-lo no navegador para ver como fica.

É importante ressaltar que, visualmente, as tags não têm estilos aplicados automaticamente. Elas servem apenas para dividir e organizar a página. A aparência, cores e fontes serão definidas posteriormente usando CSS.

Lembrando que podemos adicionar várias seções, como "notícias", "esportes" e "lazer". Basta copiar e colar a tag `<section>` para criar novas seções. Ao atualizar a página, veremos as seções sendo exibidas uma abaixo da outra. O estilo será definido posteriormente com o auxílio do CSS.

O HTML5 oferece recursos poderosos para criar layouts eficientes e organizados, permitindo uma melhor estruturação do conteúdo das páginas. A combinação do HTML5 com o CSS possibilita a criação de páginas web visualmente atraentes e bem-estruturadas. Ao explorar os recursos de layout no HTML5 e utilizar as tags apropriadas, é possível criar uma experiência de navegação mais intuitiva e agradável para os usuários.

É importante lembrar que o HTML5 é responsável pela estrutura e organização do conteúdo da página, enquanto o CSS é responsável pela estilização e aparência visual. Portanto, para obter um layout atraente e estilizado, é necessário combinar o HTML5 com o CSS, aplicando estilos, cores, fontes e outros elementos visuais.

A medida que adicionamos as tags de layout, como <header>, <nav>, <section> e <footer>, em nossa página "Oi mundo", podemos ir testando e ajustando o estilo com o CSS para alcançar o resultado desejado. Por exemplo, podemos definir estilos para o cabeçalho, como cores de fundo, tamanhos de fonte e margens. O mesmo pode ser feito para o menu de navegação, seções e rodapé.

Vale ressaltar que o HTML5 oferece flexibilidade na criação de layouts. É possível personalizar e adaptar o posicionamento, tamanho e estilo dos elementos de acordo com as necessidades e preferências do projeto. Além disso, é possível utilizar frameworks e bibliotecas CSS, como Bootstrap, para facilitar ainda mais a criação de layouts responsivos e atrativos.

Em resumo, os layouts no HTML5 permitem organizar o conteúdo de uma página de forma clara e estruturada. Utilizando as tags apropriadas, como <header>, <nav>, <section> e <footer>, é possível criar uma hierarquia visualmente compreensível para os usuários. A estilização e personalização do layout são feitas através do CSS, possibilitando uma aparência atraente e adequada ao propósito do site.

Combinando as técnicas adequadas do HTML5 e do CSS, é possível criar páginas web funcionais, visualmente agradáveis e responsivas, proporcionando uma experiência de navegação de qualidade para os usuários.
