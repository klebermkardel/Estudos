# Formatação

Este é um resumo dos conceitos apresentados sobre formatação básica de páginas HTML. Abaixo estão os principais tópicos abordados:

1. Restrições na formatação do HTML:
   - A formatação de fonte, tamanho de letra e cores é limitada no HTML.
   - A formatação de páginas HTML é feita principalmente através do arquivo de estilo CSS.

2. Elementos de formatação da página:
   - Headers (h1, h2, h3, h4, h5, h6) são utilizados para definir títulos.
   - Headers são semelhantes aos títulos no Microsoft Word, permitindo uma hierarquia visual.
   - Exemplo de uso: `<header><h1>Curso de HTML</h1></header>`

3. Quebra de linha:
   - A tag `<br>` é utilizada para criar uma quebra de linha.
   - Essa tag não requer uma tag de fechamento.
   - Exemplo de uso: `<section>Estou aprendendo sobre HTML.<br>O HTML é composto por tags</section>`

4. Parágrafos:
   - A tag `<p>` é utilizada para separar parágrafos.
   - Ela cria um espaçamento correto entre os parágrafos.
   - Exemplo de uso:
     ```
     <section>
         <p>Parágrafo 1</p>
         <p>Parágrafo 2</p>
     </section>
     ```

5. Linha separadora:
   - A tag `<hr>` cria uma linha horizontal que separa elementos.
   - Ela percorre toda a extensão do navegador.
   - Exemplo de uso: `<section><p>Linha 1</p><hr><p>Linha 2</p></section>`

6. Estilização de texto:
   - A tag `<b>` é usada para colocar texto em negrito.
   - A tag `<i>` é usada para colocar texto em itálico.
   - Exemplo de uso: `<section><p>Estou aprendendo sobre <b>HTML</b>.<br>O <i>HTML</i> é composto por tags</p></section>`

7. Comentários:
   - Os comentários são trechos de texto que não são exibidos ao usuário.
   - São úteis para adicionar informações que pertencem ao código, mas não devem ser visíveis.
   - São inseridos entre as tags `<!--` e `-->`.
   - Exemplo de uso: `<!-- Comentário -->`

É importante respeitar as boas práticas ao utilizar formatação e comentários, evitando conteúdo inadequado ou constrangedor.

Observação: Este resumo não abrange todos os aspectos do HTML e da formatação avançada, mas oferece uma introdução aos conceitos básicos. Para mais informações, consulte recursos adicionais sobre HTML.
