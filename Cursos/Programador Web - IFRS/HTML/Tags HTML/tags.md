# Lista de Tags HTML

Este repositório contém um documento PDF com uma lista de tags HTML. Essa lista é uma referência útil para desenvolvedores web, pois contém as principais tags usadas na criação de páginas HTML. É importante que você guarde essa lista para consulta sempre que necessário, seja em formato digital ou impresso.

## Uso da lista

A lista está disponível no seguinte site: [Lista de Tags HTML](https://websitesetup.org/HTML5-cheat-sheet.pdf). Ao acessar o link, você encontrará o documento PDF contendo as tags HTML.

## Tradução e Explicação

A lista está em inglês, mas se você não estiver familiarizado com algum termo, é recomendado o uso de um dicionário ou tradutor para entender a descrição de cada tag. Lembre-se de não traduzir as próprias tags, pois elas devem ser usadas apenas no formato original em inglês.

## Material de Estudo

É altamente recomendado que você mantenha essa lista de tags HTML em seus materiais de estudo, independentemente de optar por tê-la no seu computador ou em uma folha impressa. Ter acesso rápido a essa referência facilitará seu trabalho como desenvolvedor web e ajudará a criar páginas HTML corretamente estruturadas.
