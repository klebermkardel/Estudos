# Introdução ao Desenvolvimento de Páginas HTML

Este é um resumo do texto que fornece uma introdução ao desenvolvimento de páginas HTML e aborda os conceitos básicos relacionados à criação de sites na internet. O texto apresenta informações sobre o funcionamento dos sites, as linguagens de programação envolvidas e as ferramentas necessárias para começar a programar para a web.

## Resumo

- O funcionamento de um site: conjunto de páginas web exibidas nos navegadores, acessadas por meio de requisições ao servidor.
- Linguagens de programação para internet: linguagens do cliente (HTML, CSS e JavaScript) responsáveis pela apresentação e interação na página; linguagens do servidor (PHP, ASP, JSP) capazes de realizar tarefas mais complexas.
- Evolução das linguagens de programação para web: surgimento e desenvolvimento do HTML, CSS e JavaScript, avanços com o HTML 4, CSS 2 e XHTML, introdução de interatividade com componentes AJAX e documentação do HTML 5.
- O papel do HTML: linguagem de marcação para definir a apresentação das informações em uma página web.
- Ferramentas para programação para internet: a necessidade de um navegador para visualização das páginas e um editor de texto como o Notepad++ ou Sublime Text para escrever o código HTML.

## Tópicos

1. Funcionamento de um site:
   - Conjunto de páginas web processadas pelos navegadores.
   - Requisição feita pelo navegador ao servidor.
   - Processamento e retorno de uma página HTML como resposta.

2. Linguagens de programação para internet:
   - Linguagens do cliente: HTML, CSS e JavaScript.
   - Linguagens do servidor: PHP, ASP, JSP, etc.
   - Execução no navegador (cliente) ou no servidor.

3. Evolução das linguagens de programação para web:
   - Surgimento do HTML, CSS e JavaScript.
   - HTML 4 e CSS 2 com maior utilização.
   - Introdução do XHTML e mudança para Tableless Web Design.
   - Componentes AJAX e documentação do HTML 5.

4. Papel do HTML:
   - Definição da apresentação das informações de um site.
   - Interpretação pelo navegador para exibir conteúdo compreensível.

5. Ferramentas para programação para internet:
   - Navegador para visualização das páginas.
   - Editor de texto, como o Notepad++ ou o Sublime Text.
   - Testar em diferentes navegadores.
   - Editores gráficos como DreamWeaver e TinyMCE (opcional, requer mais conhecimento da linguagem).
