# Primeiros Passos

Este é um resumo do vídeo que fornece os primeiros passos no desenvolvimento de páginas HTML. O vídeo aborda a natureza do HTML como uma linguagem de marcação e explica como marcar elementos, textos, imagens e vídeos para exibi-los em uma página da web. Também são apresentados os elementos básicos necessários na criação de uma página HTML, como a tag <html>, <head> e <body>. O vídeo demonstra o uso do bloco de notas como editor de texto e como salvar o arquivo com a extensão .html. Além disso, mostra como abrir a página em um navegador e testar seu conteúdo.

## Resumo

- HTML é uma linguagem de marcação, não uma linguagem de programação.
- Os elementos básicos de uma página HTML incluem a tag <html>, <head> e <body>.
- O cabeçalho (<head>) contém informações como o título da página e arquivos complementares.
- O corpo (<body>) contém o conteúdo visual da página, como texto, imagens e vídeos.
- O desenvolvimento de uma página HTML envolve marcar os elementos desejados com tags apropriadas.
- O bloco de notas pode ser usado como um editor de texto para escrever código HTML.
- O arquivo HTML deve ser salvo com a extensão .html.
- O arquivo HTML pode ser aberto em um navegador para visualização e teste.

## Passos Iniciais

1. Abra um editor de texto, como o Bloco de Notas.
2. Digite os elementos básicos da página HTML:

```html
<html>
    <head>
       <title>Título da Página</title>
    </head>
    <body></body>
</html>
```

3. No cabeçalho, insira informações como o título da página.
4. No corpo da página, adicione elementos visuais, como texto, imagens e vídeos.
5. Salve o arquivo com a extensão .html.
6. Abra o arquivo em um navegador para visualizar a página.
7. Atualize a página para ver as modificações feitas no código.

Esses são os primeiros passos para criar e testar páginas HTML. A partir desses conceitos básicos, é possível desenvolver páginas mais complexas e interativas.
