# Git e Versionamento

## Versionamento

- Registro de mudanças em arquivos, que possibilita recuperação ou acesso a versões anteriores;

- Desenvolvimento de código em colaboração com outros integrantes.

## Git

- Git é uma sistema de versionamento de código, que guarda os registros de versão como snapshots (fotos) do estado do projeto, além da referência/caminho para essa foto.
