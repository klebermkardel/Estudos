# Santander Coders 2023 - Trilha Front End

Este repositório foi criado para armazenar todo o conteúdo de aprendizado do programa Santander Coders 2023 - Trilha Front End. Aqui você encontrará recursos, exercícios, projetos e outros materiais relacionados ao curso.

## Sobre o Santander Coders

O Santander Coders é um programa de capacitação profissional em tecnologia oferecido pelo Santander em parceria com instituições de ensino e organizações sem fins lucrativos. O programa tem como objetivo promover a formação e qualificação de talentos na área de desenvolvimento de software.

## Trilha Front End

A trilha Front End do Santander Coders 2023 é voltada para aqueles que desejam aprender e aprimorar suas habilidades no desenvolvimento de interfaces e experiências de usuário para aplicações web. Durante o curso, serão abordados os seguintes tópicos:

- HTML (HyperText Markup Language)
- CSS (Cascading Style Sheets)
- JavaScript
- Frameworks e bibliotecas populares (por exemplo, React, Angular, Vue.js)
- Práticas de desenvolvimento front end (responsive design, acessibilidade, otimização de desempenho)
