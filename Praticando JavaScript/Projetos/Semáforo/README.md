# Semáforo

Este é um projeto simples de semáforo implementado usando HTML, CSS e JavaScript. O semáforo exibe três cores principais - vermelho, amarelo e verde - e permite ao usuário controlar manualmente as cores ou ativá-las automaticamente.

## Funcionalidades

- O semáforo exibe inicialmente a cor desligada.
- Existem quatro botões disponíveis:
  - Vermelho: acende a cor vermelha do semáforo.
  - Amarelo: acende a cor amarela do semáforo.
  - Verde: acende a cor verde do semáforo.
  - Automático: alterna automaticamente as cores do semáforo em um intervalo de 1 segundo.
- Ao clicar em qualquer um dos botões de cor, a cor correspondente é ativada e qualquer modo automático é interrompido.
- O modo automático alterna automaticamente as cores do semáforo em um ciclo contínuo: vermelho, amarelo, verde. Esse ciclo é repetido indefinidamente até que o modo automático seja interrompido.

## Estrutura do Projeto

O projeto consiste em três arquivos principais:

- `index.html`: Este arquivo contém a estrutura HTML básica do semáforo, incluindo os botões e a área onde a cor é exibida.
- `style.css`: Este arquivo contém as regras de estilo CSS para posicionar e estilizar os elementos do semáforo.
- `semaforo.js`: Este arquivo contém o código JavaScript responsável por controlar as funcionalidades do semáforo.

## Uso

1. Faça o download ou clone este repositório em seu ambiente de desenvolvimento.
2. Abra o arquivo `index.html` em um navegador da web compatível.
3. Interaja com o semáforo clicando nos botões para ativar as cores manualmente ou no botão "Automático" para alternar as cores automaticamente.
