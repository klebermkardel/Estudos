# Desafio #100DaysOfCode em JavaScript

## Descrição
Este repositório foi criado para acompanhar o meu progresso no desafio #100DaysOfCode, proposto pela Giulia Bordignon. Durante os próximos 100 dias, estarei dedicando tempo diariamente para praticar e aprimorar minhas habilidades de programação em JavaScript, desde conceitos básicos até níveis mais avançados.

## Objetivos
- Escrever exercícios em JavaScript todos os dias, abrangendo desde conceitos básicos até desafios mais complexos.
- Dominar a linguagem JavaScript e aprimorar minha lógica de programação.
- Compartilhar meu progresso diário neste repositório e nas redes sociais usando as hashtags #100DaysOfCode e #Desafio100DiasDeCodigo.

## Regras
- Passar pelo menos 1 hora por dia trabalhando em exercícios de programação em JavaScript.
- Atualizar diariamente este repositório com o código desenvolvido no dia, juntamente com uma breve descrição do que foi aprendido ou praticado.
- Publicar regularmente nas redes sociais sobre o meu progresso no desafio, usando as hashtags mencionadas acima.
