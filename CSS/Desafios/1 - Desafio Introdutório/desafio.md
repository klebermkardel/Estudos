# Desafio introdutório

A melhor forma de fixar isso é escrevendo o código. Se você já tiver um editor instalado (como o VS Code), pode usar ele. Se não tiver nada pronto ainda, pode abrir um site gratuito como o Codepen.io para testar direto no navegador.

Tente escrever um código onde:

Todos os parágrafos (p) da sua página fiquem com a cor cinza (gray).
