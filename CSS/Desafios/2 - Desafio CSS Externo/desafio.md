# Desafio CSS Externo

Se você estiver usando o VS Code (ou outro editor no computador), crie esses dois arquivos na mesma pasta e teste no navegador para ver o resultado. Se estiver usando o CodePen, ele já separa o HTML e o CSS em caixas separadas automaticamente (o que simula o CSS externo).

Para exercitarmos essa nova estrutura e as propriedades de texto que vimos antes, tente criar essa mesma estrutura separada e faça o seguinte:

No HTML, crie um título ```<h1>``` e um parágrafo ```<p>```.

No seu arquivo CSS separado, mude a fonte de toda a página para ```sans-serif```. (Dica: você pode aplicar a propriedade ```font-family``` direto na tag ```body``` para que tudo dentro dela herde essa fonte).

Centralize o título e mude a cor dele usando um código Hexadecimal (escolha uma cor que goste, ex: ```#3498db ```para azul claro, ```#e74c3c ```para vermelho).

Deixe o texto do parágrafo em negrito usando a propriedade que vimos antes (```font-weight```).
