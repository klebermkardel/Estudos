# O que é o DOM?

Até hoje, todo o seu código JavaScript rodava e morria dentro do terminal preto do VS Code. No mundo real, o usuário não abre o terminal; ele abre uma página de internet, clica em botoões, preenche formulários e vê a tela mudar.

A Manipulação do DOM é a ponte que conecta o seu JavaScript ao **HTML** (a estrutura da página) e ao **CSS** (o visual da página).

**DOM** significa *Document Object Model* **(Modelo de Objeto do Documento)**.

Quando o navegador (Chrome, Edge, Safari) abre um arquivo ```.html```. ele lê aquele texto e o transforma em um **Objeto gigante** cheio de ramificações, muito parecido com os objetos que acabamos de estudar.

Como o HTML agora é um objeto para o navegador, o JavaScript consegue usar o ponto (```.```) para entrar nele, ler o que está escrito, **alterar** cores, **apagar elementos** ou **criar** coisas novas em tempo real.
