# Exercício 25 - Gerador de Cards de Usuários

**Gerador de Cards de Usuários**: Dado um array de objetos:

```js
const usuarios = [
  {nome: "Kleber", cargo: "Dev"}, {nome: "Ana", cargo: "Designer"}
];
```

Percorra o array, crie uma div com classe card para cada um, coloque o nome em um <h3> e o cargo em um <p> e jogue tudo na tela.
